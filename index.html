Welcome to MyLakeHouse

Project Summary

MyLakeHouse is a robust web application designed for efficient management and visualization of Apache Iceberg tables and monitoring of Apache Spark.

Purpose and Core Functionality

Facilitates viewing and management of Iceberg tables, including details, properties, and snapshot management.
Supports analytics dashboards to visualize the performance of user's Spark.

Key Features

  • Dashboard and Search Capabilities
  • Table Management
  • Snapshot Management
  • Maintenance and Configuration
  • Monitoring Spark Performance

Technical Specifications

  • Web Application Architecture: Built as a web application to ensure accessible, real-time data interaction.
  • Frontend
    • Using modern web technologies such as React and Vite to create a responsive and intuitive user interface.
    • Displaying the monitored performance metrics of Spark using Grafana Dashboard.
  • Backend
    • Leveraging FastAPI for efficient backend management, interfacing with Minio object store and Iceberg REST catalog for data handling.
    • Creating a spark listener to actively monitor the performance of all running applications and SQL queries.
    • Integrating the listener with PushGateway for making metrics data, Prometheus for time series data, and Grafana for visualization and analysis.

User Interaction and Accessibility

The website is designed to support various user interactions with the data, including exploration of data files and configurations. It is integrated with Grafana to provide a comprehensive dashboard experience with Spark monitoring. Features such as the search tool have been enhanced with filtering options to streamline user workflows.
