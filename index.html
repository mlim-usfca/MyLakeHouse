Welcome to MyLakeHouse

Project Summary

MyLakeHouse is a robust web application designed for efficient management and visualization of Apache Iceberg tables and monitoring of Apache Spark.

Purpose and Core Functionality

Facilitates viewing and management of Iceberg tables, including details, properties, and snapshot management.
Supports analytics dashboards to visualize the performance of all Spark queries.

Key Features

  • Dashboard and Search Capabilities
  • Table Management
  • Snapshot Management
  • Maintenance and Configuration

Technical Specifications

  • Web Application Architecture: Built as a web application to ensure accessible, real-time data interaction.
  • Frontend: Uses modern web technologies to create a responsive and intuitive user interface.
  • Backend: Leverages FastAPI for efficient backend management, interfacing with Minio object store and Iceberg REST catalog for data handling.

User Interaction and Accessibility

Designed to support various user interactions with the data, including explorations of data files and configurations.
Integrated with Grafana for a full dashboard experience with spark monitoring. Features like the search tool are enhanced with filtering to streamline user workflows.
